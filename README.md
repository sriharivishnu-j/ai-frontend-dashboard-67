# AI-Driven Real-time Dashboard

This project is a real-time dashboard built with the T3 Stack (Next.js, Prisma, tRPC), focusing on AI-driven data updates and animations with Framer Motion.

## Features

- **Real-time Data:** Fetches and displays data in real-time.
- **Responsive Design:** Built using Tailwind CSS for a premium look and feel.
- **Animations:** Utilizes Framer Motion for smooth transitions and animations.

## Setup

1. **Clone the repo:**

   ```bash
   git clone <repo-url>
   cd ai-dashboard
   ```

2. **Install dependencies:**

   ```bash
   yarn install
   ```

3. **Generate Prisma Client:**

   ```bash
   npx prisma generate
   ```

4. **Run the development server:**

   ```bash
   ./run.sh
   ```

5. **Access the app:**

   Open [http://localhost:3000](http://localhost:3000) in your browser.

## Testing

- Ensure the development server is running and navigate to the home page to see the dashboard in action.
